## ✨ Features
- 🛒 Full e-commerce — cart, orders, COD & Razorpay payments
- 🤖 AI Herb Advisor powered by Claude AI (symptom → herb recommendation)
- 📦 Real-time inventory & admin dashboard
- 👤 Customer dashboard — order history, wishlist, health profile
- 🔔 Push notifications & monthly subscription box
- 📊 Sales analytics with revenue charts
- 📱 Fully responsive — mobile first design
- 🔥 Firebase + Firestore backend
- 🌿 66+ herbal products with live Firestore catalog

---

## 🛠️ Tech Stack
| Layer | Technology |
|-------|-----------|
| Frontend | HTML, CSS, Vanilla JavaScript (ES6+) |
| Backend | Firebase Firestore, Firebase Auth |
| Payments | Razorpay (live), COD |
| AI | Claude API (Anthropic) |
| Hosting | Firebase Hosting |
| Auth | Google OAuth, Phone OTP |

## 📁 Project Structure
```
natures-heal/
├── index.html          # Main SPA entry point
├── css/
│   ├── main.css        # Layout, grid, sidebar
│   └── components.css  # Cards, PDP, modals, chat
├── js/
│   ├── firebase.js     # Firebase SDK + Firestore helpers
│   ├── auth.js         # Auth state, user UI
│   ├── app.js          # Catalog, filters, AI advisor
│   ├── cart.js         # Cart, orders, payments
│   ├── admin.js        # Admin panel, inventory
│   └── features.js     # Dashboard, notifications, subscriptions
├── firebase.json       # Hosting + CSP headers
└── firestore.rules     # Security rules
```
